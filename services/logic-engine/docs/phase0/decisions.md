# Phase 0 Decision Lock

Date: 2026-03-17

## Architecture Decisions
1. Golden skill anatomy is immutable for this implementation sprint.
   - No edits to `skills/*/SKILL.md` frontmatter are allowed.
   - Product-serving metadata must be sidecar data generated by the logic-engine service.
2. Access scope is internal-only.
   - Only internal AIOS agents/tenants are in scope.
   - Public and external tenant routes remain deferred.
3. Execution mode for this sprint is managed-only.
   - Runtime mode is locked to `MANAGED`.
   - Client-side execution and JIT disclosure are deferred.
4. Secret strategy for this sprint.
   - Continue using current Vault and environment secret injection patterns.
   - Google Secret Manager integration is deferred.
5. Phase 3 evidence policy is locked.
   - Success runs retain metadata only.
   - Failed/blocked runs retain redacted diagnostics for 30 days.
   - Disclosure and audit metadata retain for 180 days.
   - Retention purge runs daily.
