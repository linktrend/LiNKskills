# API Specs

## Predictive Intent Log Structure
- `service`
- `action`
- `resource_id`
- `status`: `intent`
- `summary`

## Coverage Scope
- Google Workspace write actions via `gws`.
- `ltr` write actions for non-Workspace Google and non-Google lanes.
- n8n workflow mutation actions (create/activate/trigger/deactivate).
